# Usage

Requirementで列挙したライブラリなどのインストール方法を説明する

```bash
git clone https://github.com/fuki01/foodManager.git
cd foodManager
bundle install --path vendor/bundle
bundle exec rails db:create
bundle exec rails db:migrate
bundle exec rails s
```
